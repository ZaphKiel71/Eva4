Documentación de la API

Ingreso y Consulta de Datos

En esta parte procederemos a explicar cómo ingresar o consultar datos.

Método Register

Para empezar, se explicará el método de Register. Se deberá ingresar en la siguiente URL en Thunderclient: http://localhost:8000/register/. El método HTTP que se debe usar es POST, donde se ingresarán los parámetros de username y password en formato JSON. Los usuarios creados anteriormente son: Felix, pedro, jose, benjamin, manuel. La contraseña para todos los usuarios es inacap123.

Método Login

Luego está el método Login. Se deberá ingresar en la URL: http://localhost:8000/login/. El método HTTP que se deberá usar es POST, ingresando nuevamente los parámetros de username y password en formato JSON. Al ingresarlo, se le entregará el token del usuario que se usará más adelante.

Método Profile

El método Profile sirve para ver los datos del usuario, menos la contraseña. Para ingresar, debe usar la URL: http://localhost:8000/profile/ con el método POST. Para usar este método, deberá usar el token que se entregó anteriormente. Este lo deberá ingresar en los Headers de Thunderclient. En la parte de header, deberá ingresar Authorization y en value ingresará de esta forma: Token (ingresas tu token).

Método Products

El método Products sirve para listar todos los productos. Deberá ingresar a la siguiente URL: http://localhost:8000/products/ con el método HTTP de GET.

Método Create Products

El método Create Products sirve para crear un nuevo producto. Para ingresar, deberá usar la siguiente URL: http://localhost:8000/products/create/. El método HTTP que se usará será GET, y deberá ingresar los parámetros de nombre, precio y stock en formato JSON. Para usar este método, deberá usar el token que se entregó anteriormente. Este lo deberá ingresar en los Headers de Thunderclient, en la parte de header, deberá ingresar Authorization y en value ingresará de esta forma: Token (ingresas tu token).

Método Actualizar Productos

El método Actualizar Productos se usa para actualizar un producto. Para usarlo, deberá ingresar en la siguiente URL: http://localhost:8000/products/pk/update/, donde pk será el ID del producto que lo puedes obtener desde la página o el método Products. El método HTTP que usará es PATCH, y se deberán ingresar los parámetros de name, price y stock nuevos en formato JSON. Para usar este método, deberá usar el token que se entregó anteriormente. Este lo deberá ingresar en los Headers de Thunderclient, en la parte de header, deberá ingresar Authorization y en value ingresará de esta forma: Token (ingresas tu token).

Método Destroy Product

El método Destroy Product es para eliminar un producto. La URL es: http://localhost:8000/products/pk/delete/ con el método HTTP DELETE, donde pk es el ID del producto. Para usar este método, deberá usar el token que se entregó anteriormente. Este lo deberá ingresar en los Headers de Thunderclient, en la parte de header, deberá ingresar Authorization y en value ingresará de esta forma: Token (ingresas tu token).

Método Order List

El método Order List sirve para listar las órdenes. Para usarlo, se usará la siguiente URL: http://localhost:8000/products/orders/list/. Se usará el método HTTP de GET.

Nota: Esta funcionalidad no está terminada.

Método Create Order

El método Create Order sirve para crear una orden.para ingresar se usara la http://localhost:8000/product/orders/create/ Se usará el método HTTP de POST. Para usar este método,deberas ingresar el id del producto, el precio y la quantity en formato json, deberás usar el token que se entregó anteriormente. Este lo deberá ingresar en los Headers de Thunderclient, en la parte de header, deberá ingresar Authorization y en value ingresará de esta forma: Token (ingresas tu token).

Nota: Esta funcionalidad no está terminada.
